# SnakeWar
Multiplayer snake game

# Features
- Snake Game
- Webserver

# To Come:
- multiplayer support
- database

# How to install and run?
- clone the repo
- go into `webserver`
- build docker image manually or use `build.bat`
- run container manually or with `start.bat`
- copy the Server-Url into your browser and enjoy some SNAKE :)
